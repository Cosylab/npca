Class EPICSPlugin

Object
   |
   +--EPICSPlugin
class EPICSPlugin


EPICSPlugin is a scriptable object implemented as an NPAPI plugin. To obtain EPICSPlugin object use: 

    // inside HTML body
    <embed id="EPICSPlugin" type="application/mozilla-npca-scriptable-plugin" width="0" height="0"/>

    // inside JavaScript section 
    // get scriptable object
    var epicsPlugin = document.getElementById("EPICSPlugin");

pendEvents

int pendEvents(<int> eventsToProcess)
    Handle events (callbacks). This method is an equivalent to EPICS CA context.pendEvent() method with non-preemptive CA context. If this method is not called no events (callbacks) will be called by the plugin.
    Usage: 
        function handleEvents() {
      epicsPlugin.pendEvents();
      
      // 10Hz
      setTimeout(function() { handleEvents(); }, 100);
    }
    
    // start handling events
    handleEvents();
    Parameters:
      eventsToProcess - maximum number of events to process. This parameter is optional, if not specified default value 16 will be used.
    Returns:
      actual number of events processed.
